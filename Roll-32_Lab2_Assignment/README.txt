`Running The Program`
1. Run the Program.cpp/Program.exe program
2. Choose whether to encode or decode
3. Choose Scheme

`For encoding`: Change the encoder_input.txt file. Format: (101001) 
	      You can see the output in the console and the generated encoder_output.txt file.

`For decoding`: Change the decoder_input.txt file. Format: (+5V, -5V, +5V, -5V, -5V,), In Case 
                of AMI, Format: (+5V, 0V, -5V,)
	      You can see the output in the console and the generated decoder_output.txt file.
